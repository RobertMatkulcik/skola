4. Trigger in Languages – Teoria o dostupných jazykoch zameraných na tvorbu triggerov + instalacia

  • O programovaní na strane servera
    • PL/Python
    • PL/pgSQL
      • https://www.postgresql.org/docs/9.3/static/plpgsql.html
    • PL/TCL
    • PL/PEARL
      Functions can be written in:
    • C
    • SQL
  • A procedural language (PL) Dozens of PLs, some niche:
    • PL/R
    • PL/OpenCL
    • PL/LOLCODE
  • Some mainstream:
    • PL/PgSQL • PL/Python
