Trigger

Použitie triggera v jazyku PL/Python

Vytvorenie triggera:

CREATE OR REPLACE FUNCTION oznam_po_zavolani()
RETURNS TRIGGER
AS $$
plpy.notice('Trigger bol spustený!')
$$ LANGUAGE plpythonu;

Tabuľky:

CREATE TABLE tabulka(id int);

Zavolanie triggera:

CREATE TRIGGER tabulka_oznam BEFORE INSERT ON tabulka EXECUTE
PROCEDURE oznam_po_zavolani();

Pred pokusom o INSERT dát trigger zavolá funkciu a plpy.notice vypíše nastavený text:

INSERT INTO ttable VALUES(1);
NOTICE: Trigger bol spustený!
CONTEXT: PL/Python function "oznam_po_zavolani"
INSERT 0 1
